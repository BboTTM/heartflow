# 关系推进.skill 录入问题

## archetype 模式

一次只问一个：

1. 这个人怎么称呼
2. 你们现在是什么阶段
3. 你想用什么关系风格 archetype
4. 星座象限或星座是什么
5. MBTI 是什么

如果用户说“我们已经认识、也聊了一阵”，可以提示多数实战演练从 `升温` 开始更合适。

## real-person 模式

一次只问一个：

1. 这个人怎么称呼
2. 你们现在是什么阶段
3. 你准备提供什么资料
4. 你主观觉得对方是慢热、主动、边界感强，还是别的什么

如果用户说“我们已经认识、也聊了一阵”，可以提示多数实战演练从 `升温` 开始更合适。
